## Ambientes

Nuestros clientes pueden acceder a dos ambientes: staging y producción.

**Portal Staging:**`https://portal.stg.hellopay.com.co` 

**API Staging**: `https://api.stg.hellopay.com.co`

**Portal Producción**:`https://portal.hellopay.com.co` 

**API Producción**: `https://api.hellopay.com.co`

## Autenticación

Obten tus API Keys:

1. Ingresa al portal de HelloPay. ([Staging ](https://portal.stg.hellopay.com.co)| [Producción](https://portal.hellopay.com.co))
2. Ve a "Configuraciones -\> API Keys. ([Staging](https://portal.stg.hellopay.com.co/settings/api-keys) | [Producción](https://portal.hellopay.com.co/settings/api-keys))
3. Click **Crear API Key**.
4. Guarda tu **API Key** (solo estará visible al momento de ser creada).
